B. OVERVIEW
A robust backend API for a Car Rental Marketplace. This system handles the complex logic of vehicle management, customer bookings, and data transformations. Built with a clean architecture pattern to ensure scalability and ease of maintenance.

C. FEATURES
- Car Management: Full CRUD operations for vehicle listings.
- Data Mapping: Uses DTOs (Data Transfer Objects) and Mappings to ensure secure and efficient data transfer.
- Booking Logic: Managed through dedicated Services to handle business rules.
- Database Integration: Data layer powered by Entity Framework Core.

D. TECH STACK
- Framework: .NET Core API
- Language: C#
- Architecture: Controller-Service-Repository Pattern
- Tools: Vite + React (for Frontend)

E. GETTING STARTED
- To get your development environment up and running, follow these steps:
- Install dependencies
- Bash
- npm install
- Start the development server
- Bash
- npx expo start
- Run the Backend
- Open RentCar.Api.sln in Visual Studio.
- Press F5 or run dotnet run.
